# In Your Own Words

**Website:** [https://inyourownwords.github.io/](https://inyourownwords.github.io/)

This is the project website for *In Your Own Words: Computationally Identifying Interpretable Themes in Free-Text Survey Data*.

The site includes an interactive theme explorer, a description of the data collection and computational pipeline, and a form to contribute your own free-text identity description.

For questions or information about the dataset or methods, please contact Jenny S. Wang (jewang@hbs.edu).
